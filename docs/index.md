## 30 秒启动验证服务

> 💡 复制下面的命令，体验一下验证有多简单

```bash
# 全局安装
npm install -g ai-verify-mcp

# 健康检查（验证环境是否就绪）
ai-verify-mcp health

# HTTP 模式启动（默认端口 3000）
ai-verify-mcp http --port 3000
```
